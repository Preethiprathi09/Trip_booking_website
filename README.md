# Trip_booking_website
To Coorg Today is a modern travel website designed to promote tourism in Kodagu (Coorg). The website focuses on delivering a luxury and immersive browsing experience through subtle animations, elegant typography, and polished layout design.  This project was built as a portfolio piece to demonstrate front-end development skills and UI refinement.
# 🌿 To Coorg Today – Luxury Travel Website

A premium, interactive travel website inspired by luxury hospitality design.

## ✨ Features
- Elegant frosted-glass carousel with animated captions
- Smooth scroll and reveal animations
- Luxury-themed UI with Playfair Display typography
- Interactive hover effects on cards
- Premium booking form layout
- Scroll progress indicator
- Transparent navbar with dynamic scroll behavior

## 🛠 Tech Stack
- HTML5
- Bootstrap 5
- Custom CSS Animations
- JavaScript (Scroll Effects)

## 🎯 Purpose
Built as a portfolio project to demonstrate front-end design skills, UI refinement, and interactive user experience.
